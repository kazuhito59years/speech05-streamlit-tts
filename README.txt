Speech05 - 読み上げアプリ（履歴保存＋複数読み上げ対応）

====================================
■ 概要
====================================
テキストを音声に変換するアプリです。
複数行の一括読み上げ、履歴保存、音声選択に対応しています。

====================================
■ 主な機能
====================================
・テキスト読み上げ（Text-to-Speech）
・複数行の一括読み上げ（改行区切り）
・音声選択（alloy, nova, shimmer など）
・音声ファイル保存
・履歴保存（JSON）
・履歴から再生・ダウンロード

====================================
■ 動作環境
====================================
・Python 3.10 以上推奨
・インターネット接続

====================================
■ セットアップ手順
====================================

① 仮想環境作成（推奨）
------------------------------------
python -m venv venv
source venv/bin/activate   （Linux / Mac）
venv\Scripts\activate      （Windows）

② ライブラリインストール
------------------------------------
pip install -r requirements.txt

③ OpenAI APIキーの取得
------------------------------------
OpenAIのサイトでAPIキーを取得してください。

https://platform.openai.com/

④ .envファイル作成
------------------------------------
プロジェクト直下に .env ファイルを作成し、以下を記述

OPENAI_API_KEY=あなたのAPIキー

⑤ アプリ起動
------------------------------------
streamlit run app.py

====================================

Speech05/

====================================

・長文や大量処理時はコストに注意してください
・history.json が肥大化する可能性があります
